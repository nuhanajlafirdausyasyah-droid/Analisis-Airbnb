# Analisis Airbnb Eropa
Analisis sederhana berdasarkan dataset Airbnb Prices in European Cities.

## Metode
- Data: 51,707 listing dari 10 kota Eropa
- Analisis: Segmentasi klaster, perbandingan weekend-weekday

## Hasil Utama
1. Ditemukan 4 segmen pasar utama
2. Weekend lebih mahal untuk semua segmen
3. Rekomendasi berbeda untuk setiap klaster

- Dataset asli: https://www.kaggle.com/datasets/thedevastator/airbnb-prices-in-european-cities
